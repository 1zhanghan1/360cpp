# 实现C++中间件

## 线程池

功能：

- 手动设置线程池线程数量
- 可选的线程池初始化策略：懒加载，饥饿式加载
- 可选的任务队列满操作：拒绝，异常
- 可选的底层任务队列实现：阻塞队列，阻塞环形数组，无锁队列，无锁环形数组
- 支持线程池中线程数动态控制，可设置的最大线程数、空闲、忙碌阈值
- 异常安全，提交任务抛出异常不会影响线程池正常运行
- 支持二级优先级任务，单线程执行优先任务
